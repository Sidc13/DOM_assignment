# Student Registration System

A simple web-based student registration system built using **HTML**, **CSS**, and **JavaScript**.
The system allows users to:

- Add student details (Name, Roll Number, Class, Phone)
- Display registered students in a list below the form
- Edit or delete any student record
- Data remains saved using localStorage (prevents loss after refresh)
